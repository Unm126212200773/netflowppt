NetFlow Application

Network Traffic Analysis

  • Developed by Cisco
  • Flow-based monitoring
  • Enterprise & ISP usage

What is NetFlow?

  • Collects traffic metadata
  • No packet payload
  • Lightweight & scalable
  • Flow-based analysis

Why NetFlow Matters

  • Traffic visibility
  • Detect abnormal usage
  • Identify top talkers
  • Capacity planning

NetFlow Fundamentals

  • Traffic grouped as flows
  • Unidirectional flows
  • Stored in flow cache
  • Exported after timeout

Flow Definition

  • Source IP
  • Destination IP
  • Ports & protocol
  • Bytes & packets

NetFlow Architecture

  • Exporter (Router/Switch)
  • Collector (Storage)
  • Analyzer (Reports)

Flow Lifecycle

  • First packet arrives
  • Flow created
  • Flow updated
  • Exported to collector

NetFlow Versions

  • NetFlow v5
  • NetFlow v9
  • IPFIX (Standard)

Applications

  • Traffic monitoring
  • DDoS detection
  • Security analysis
  • Troubleshooting

Summary

  • Improves visibility
  • Lightweight monitoring
  • Essential for networks
